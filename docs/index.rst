Akrophonobolos
==========================================

Akrophonobolos is a Python package for handling Ancient Athenian
monetary values in talents, drachmas and obols, including input and
output in Greek acrophonic numerals (such as ğ…, ğ…, ğ…Œ, ğ…‹, ğ…Š, etc.)
